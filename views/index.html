Timestamp Microservice

Description

The Timestamp Microservice accepts a date in unix time(milliseconds elapsed since ), or natural language then the Timestamp Microservice outputs the unix time and the natural language time regardless of input format.

How To Use

Append your input to the end of the current url, like so: https://rapid-horn.glitch.me/52222222223
or https://rapid-horn.glitch.me/august 28 1971
This can either be done in your address bar or by using the http GET protocol.

Example output: {"unix":12222222, "natural":"January 1, 1970"}
